Lab.HTML

Attribute

ชื่อ Tag Attribute การใช้งาน
<hr> width, align, size, noshade, color เส้นคั่นแนวนอน
<p> align ใช้สำหรับจัดแบ่งย่อหน้า
<div> class, id สำหรับแบ่งส่วนต่างๆ ภายใน document
<img> alt, crossorigin, height, ismap, loading, longdesc, referrerpolicy, sizes, src, srcset, usemap, width แสดงรูปภาพกราฟิก
<a> download, href, hreflang, media, ping, referrerpolicy, rel, target, type กำหนดจุดลิงก์
<ul> type แสดงรายการข้อมูลแบบแจกแจงเป็นข้อๆ โดยไม่เรียงลำดับ
<ol> type, start, reversed แสดงรายการข้อมูลแบบแจกแจงเป็นข้อๆ โดยเรียงลำดับ
<table> align, border, bordercolor, bgcolor, width, valign, cellspacing, cellpadding, colspan, rowspan สร้างตาราง
<form> accept-charset, action, autocomplete, enctype, method, name, novalidate, rel, target สร้างแบบฟอร์มป้อนข้อมูล
<button> autofocus, disabled, form, formaction, formenctype, formmethod, formnovalidate, formtarget, name, type, value สร้างปุ่มกดในแบบฟอร์ม

Differentiation Tag

ชื่อ Tag ความเหมือน ความแตกต่าง
<hr> & <br> ขึ้นบรรทัดใหม่ <hr> คือ เส้นคั่นแนวนอน
<br> เป็นคำสั่งที่กำหนดจุดสิ้นสุดของบรรทัด ขึ้นบรรทัดใหม่
<ul> & <ol> ลิสต์ (List) <ul> เรียงลำดับเป็นสัญลักษณ์
<ol> เรียงลำดับเป็นตัวเลข
<select> & <checkbox> & <radio button> เลือกตอบคำตอบที่มีไว้อยู่แล้ว <select> เลือกตอบแบบ drop-down list
<checkbox> ผู้ใช้งานสามารถเลือกตัวเลือกได้มากกว่าหนึ่งตัวเลือกจากตัวเลือกทั้งหมด
<radio button> ผู้ใช้งานสามารถได้เพียงตัวเลือกเดียวจากชุดของตัวเลือกทั้งหมด
<div> & <span> การจัดกลุ่มข้อความ <div> เป็น block เมื่อมี div ใหม่มา มันก็จะขึ้นบรรทัดใหม่
<span> เป็น inline ทำงานบรรทัดเดียว
<script> & <style> เป็นการใช้ภาษาอื่นใน ร่วมกับ HTML <script> javascript <style> css <script> เพื่อให้เว็บไซต์ของเราดูมีการเคลื่อนไหว สามารถตอบสนองผู้ใช้งานได้มากขึ้น โดยการใส่ Javascript ลงในโค๊ด HTML
<style> เป็นการกำหนดค่าคุณสมบัติและรูปแบบของ tag ในเอกสาร

Input Type

Input Type ความแตกต่าง
"taxt" จะกำหนดการป้อน ข้อความ 1 บรรทัด
"button" กำหนดปุ่มคลิกได้
"checkbox" ผู้ใช้งานสามารถเลือกตัวเลือกได้มากกว่าหนึ่งตัวเลือกจากตัวเลือกทั้งหมด
"radio" ผู้ใช้งานสามารถได้เพียงตัวเลือกเดียวจากชุดของตัวเลือกทั้งหมด
"password" กำหนดการป้อนรหัสผ่าน ตัวอักษรในกรอบรหัสผ่านจะแสดงเป็นจุด
"color" กำหนดตัวเลือกสี
"date" กำหนดควบคุมวันที่ (ปีเดือนและวัน (ไม่มีเวลา))
"datetime" ประเภทการป้อนข้อมูลวันที่และเวลาได้ถูกลบออกจากมาตรฐาน HTML ใช้วันที่และเวลาท้องถิ่นแทน
"datetime-local" กำหนดวันที่และเวลาการควบคุม (ปีเดือนวันชั่วโมงนาทีวินาทีและเสี้ยววินาที (โซนเวลา)
"email" กำหนดเขตข้อมูลที่อยู่อีเมล
"file" กำหนดข้อมูลไฟล์และเลือกปุ่ม "Browse ... " (สำหรับการอัปโหลดไฟล์)
"hidden" กำหนดช่องใส่ที่ซ่อนอยู่
"image" กำหนดภาพเป็นปุ่มส่ง
"month" กำหนดเดือนและปี Control (ไม่มีโซนเวลา)
"number" กำหนดช่องสำหรับป้อนตัวเลข
"range" กำหนดควบคุมสำหรับการป้อนจำนวนที่มีค่าที่แน่นอนไม่ได้เป็นสิ่งที่สำคัญ (เช่นการควบคุมเลื่อน)
"reset" กำหนดปุ่มรีเซ็ต (รีเซ็ตค่าทุกรูปแบบไปเป็นค่าเริ่มต้น)
"search" กำหนดฟิลด์ข้อความสำหรับการป้อนสตริงการค้นหา
"submit" กำหนดปุ่มสำหรับส่งข้อมูลไปยังเซิฟเวอร์ โดยข้อมูลจะส่งไปยังไฟล์ที่กำหนดในแอตทริบิวต์ action
"time" กำหนดควบคุมสำหรับการป้อนเวลา (ไม่มีโซนเวลา)
"url" กำหนดฟิลด์สำหรับป้อน URL
"week" กำหนดสัปดาห์และปี Control (ไม่มีโซนเวลา)
